# 隐私政策 (Privacy Policy)

**生效日期：2026年1月23日**

InkControl ("我们") 非常重视您的隐私。本隐私政策说明了简苹果 (simpleapples) 开发的 macOS 应用程序 **InkControl** ("本应用") 如何收集、使用和披露您的信息。

## 1. 我们收集的信息

### 1.1 个人信息的收集
为了验证您的软件许可证，我们需要收集并处理您的 **电子邮件地址**。该信息仅用于将唯一的许可证密钥与您的身份绑定，确保您购买的权益得到保护。除了电子邮件地址外，InkControl **不会** 请求或存储您的其他个人身份信息 (PII)，例如您的姓名、地址或电话号码。

### 1.2 设备和使用数据
本应用主要在您的本地设备上运行。我们不会将您的屏幕内容、使用习惯或连接的显示器信息上传到任何远程服务器。

### 1.3 许可证与试用数据
为了验证您的试用资格或软件许可证，本应用可能会在您的本地设备（如文件系统或系统偏好设置）中存储加密的许可证密钥、试用开始日期和激活状态。这些数据仅用于应用内的功能授权验证。

## 2. 数据的存储与安全

所有的配置数据、许可证信息和应用设置均存储在您的 **本地设备** 上。我们未运营任何用于收集用户数据的云端数据库。

## 3. 第三方服务

### 3.1 支付与授权
如果您购买了本应用的付费版本，交易过程由第三方支付处理商（如 Stripe）及我们的许可证分发平台处理。在交易过程中，这些平台可能会收集您的支付信息和电子邮件地址，隐私政策请参阅相应服务提供商的声明。我们仅接收生成的许可证密钥及相关的激活状态信息，不接触您的信用卡号等敏感支付信息。

### 3.2 自动更新
本应用可能通过 GitHub Releases 检查新版本。此过程遵循 GitHub 的隐私政策。

## 4. 政策变更

我们可能会不时更新本隐私政策。任何更改将在本页面上发布，更新后的政策一经发布即刻生效。建议您定期查看本政策以了解任何更改。
